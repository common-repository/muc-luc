== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Muc luc'
3. Activate Muc luc from your Plugins page.

=== Manually ===

1. Upload the `muc-luc` folder to the `/wp-content/plugins/` directory
2. Activate the Muc luc plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
